# Учебный проект "Путешествие"

## Описание проекта

Учебный проект, разработанный в рамках обучения на курсе "React + Redux" от обучающей платформы "PurpleSchool".

## Разработка

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск проекта

```bash
npm run dev
```

### 3. Сборка проекта

```bash
npm run build
```

## Технологии

- React
- Vite
